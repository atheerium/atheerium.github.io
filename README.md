# atheerium.github.io

Personal website and knowledge base.

Built with [Astro](https://astro.build), deployed on [GitHub Pages](https://pages.github.com).

## Structure

- `src/content/articles/` — Evergreen writing on domains, freelancing, automation
- `src/content/research/` — Original market analysis and data studies
- `src/content/domains/` — Domain investing portfolio and case studies
- `src/content/projects/` — Tools and systems I've built
- `src/content/notes/` — Quick notes and how-to guides

## Development

```bash
pnpm install
pnpm dev
pnpm build   # verify build before pushing
```

## Deployment

Every push to `main` auto-deploys via GitHub Actions.
