## Deployment

- Trigger: push to `main` branch
- CI: GitHub Actions (`.github/workflows/deploy.yml`)
- Build: `pnpm install --frozen-lockfile` → `pnpm build`
- Deploy: `actions/deploy-pages@v4`
- Host: GitHub Pages
- Domain: atheerium.github.io
- Build time: ~30s
- Pages built: 42

## Notes for Agents

1. Always run `pnpm build` before pushing — broken builds block the live site.
2. New domain pages go in `src/content/domains/{slug}.md`, not `src/pages/`.
3. Portfolio overview in `portfolio.md` must stay in sync with individual domain pages.
4. If adding a nav link, update BOTH the `<nav>` and `<footer>` in `Layout.astro`.
5. Em dashes (`—`) break the YAML frontmatter parser. Use `--` inside `---` blocks.
6. This file (STATE.md) should be updated when significant content is added.
